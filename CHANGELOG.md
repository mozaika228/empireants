# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-04-07

### Added
- Deterministic simulation core with configurable ACO strategies.
- Runtime supervision layer with mailbox backpressure and recovery counters.
- Realtime observability endpoint (`/metrics`, `/healthz`) and Prometheus-formatted exports.
- Scientific validation matrix with scenario-strategy KPI reporting.
- Realtime web dashboard UI for runtime and colony telemetry.
- Engineering quality gates: CI, templates, ownership, formatting/lint standards.
- Product stack packaging: Dockerfile, docker-compose, Prometheus and Grafana provisioning.
