# EmpireAnts v0.1.0

Release date: 2026-04-07

## Highlights

- Real scaling profiles (`10k`, `100k`, `1m`) with benchmark runner.
- Professional observability stack:
  - Prometheus-compatible metrics
  - live `/metrics` and `/healthz`
  - runtime latency and supervision counters
- Supervised actor runtime with backpressure and controlled recovery.
- Scientific validation matrix (`scenario x strategy`) with reproducible KPI CSV.
- Realtime web UI dashboard with trend charts and runtime diagnostics.
- Product-ready deployment package:
  - `Dockerfile`
  - `docker-compose`
  - Prometheus and Grafana provisioning
  - `Makefile` operations shortcuts
- Engineering quality and governance:
  - CI quality gates
  - PR and issue templates
  - CODEOWNERS
  - pinned toolchain and lint config
- Research documentation layer:
  - architecture spec
  - experiment protocol
  - reproducibility guide
  - KPI glossary
  - ADR index and accepted decisions

## Core commands

```bash
make up
make validate
cargo run --release --bin scale_benchmark
cargo run --release --bin scientific_validation
```
