# Roadmap

## Near-term

- Live control plane: parameter edits over HTTP/WebSocket without restart.
- Replay mode: deterministic event logs and scenario playback.
- Extended dashboard pack: biological validation and ACO comparison views.

## Mid-term

- Multi-shard runtime across CPU worker pools.
- GPU compute path for pheromone diffusion and neighborhood scoring.
- Baseline biological datasets and validation notebooks.

## Long-term

- Distributed colony simulation across multiple machines.
- Fault-tolerant supervisor tree with cross-node actor migration.
- Published benchmark suite with versioned performance baselines.
